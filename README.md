# Basta Desserts

This repository contains a small dessert website. The original static pages remain in the project root.
A React-based version lives in the `react-app` folder. The `react-app/styles` directory contains copies of the original CSS files so the React pages share the same styling.

## Running the React version

1. Open a terminal in the `react-app` folder.
2. Run `npm install` to install the local dev server.
3. Start the site with `npm start` and open the provided URL in your browser (or let VS Code open it automatically).

You can also simply open `react-app/index.html` directly if you prefer.
